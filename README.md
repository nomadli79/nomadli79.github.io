## 环境及开发
01. cd git path
02. npm install hexo --save
03. hexo -v
04. hexo init
05. npm install
06. hexo g
07. npm install hexo-deployer-git --save
08. hexo new
09. hexo clean
10. hexo d -g
11. hexo s
12. hexo deploy
13. Front-matter
